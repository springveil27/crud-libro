# CRUD de Libros con PHP, MySQL y JavaScript

Proyecto de práctica para gestionar libros (Crear, Leer, Actualizar, Eliminar) usando:

- PHP como backend (API REST básica).
- MySQL como base de datos.
- HTML, CSS y JavaScript para el frontend.

El proyecto sigue la metodología Git Flow con ramas main, dev, qa y features.
